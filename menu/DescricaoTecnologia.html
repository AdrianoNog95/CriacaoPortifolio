Nome tecnologia 1: Embarcadero Delphi XE5
Site oficial da tecnologia:https://www.embarcadero.com/br/products

Definição da tecnologia:Embarcadero Delphi, anteriormente conhecido como CodeGear Delphi, Inprise Delphi e Borland Delphi, também conhecido como Delphi, é um compilador, uma IDE e uma linguagem de programação, produzido anteriormente pela Borland Software Corporation e atualmente pela Embarcadero. O Delphi, originalmente direcionado para a plataforma Windows, chegou a ser usado para desenvolvimento de aplicações nativas para Linux através do Kylix (o Kylix é um IDE para as linguagens C++ e Object Pascal), e para o framework Microsoft .NET em suas versões mais recentes. O desenvolvimento do Kylix foi descontinuado. O Delphi é muito utilizado no desenvolvimento de aplicações desktop, aplicações multicamadas e cliente/servidor, compatível com os bancos de dados mais conhecidos do mercado. O Delphi pode ser utilizado para diversos tipos de desenvolvimento de projeto, abrangendo desde Serviços a Aplicações Web e CTI. O nome Delphi é inspirado na cidade de Delfos, o único local na Grécia antiga em que era possível consultar o Oráculo de Delfos. O nome deve-se ao fato de que os desenvolvedores do compilador procuravam uma ferramenta capaz de aceder ao banco de dados Oracle - originando a frase: "a única maneira de aceder ao oráculo é usando Delphi".

3 prós de se usar esta tecnologia:
1ª A existência de uma grande quantidade de componentes prontos em sua biblioteca;
2ª Facilidade de uso e aprendizado;
3ª Desenvolvimento rápido para criação de telas para sistemas.

3 Limitações:
1ª A cada versão lançada do Delphi, a empresa tenta manter a retrocompatibilidade, para que seus utilizadores consigam reutilizar códigos desenvolvidos em versões anteriores, porém as vezes os novos recursos tornam novas versões menos compatíveis;
2ª Exemplo: as bibliotecas padrão VCL/RTL se tornaram cada vez mais restritivas;
3ª Algumas versões do Delphi receberam reclamações por instabilidade.

6 grandes projetos que utilizam a tecnologia:Göpel eletronic; TachoEASY; Sticky Password; Syntax planbar; OpusCapita; HeidiSQL.

Nome tecnologia 3: Weather App
Site oficial da tecnologia:https://www.javascript.com/

Definição da tecnologia:JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web — toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática — mostrando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animados, etc. — você pode apostar que o JavaScript provavelmente está envolvido. É a terceira camada do bolo das tecnologias padrões da web, duas das quais (HTML e CSS) nós falamos com muito mais detalhes em outras partes da Área de Aprendizado.

3 prós de se usar esta tecnologia:
1ª Velocidade: Como JavaScript é uma linguagem 'interpretada', reduz o tempo exigido por outras linguagens de programação como Java para compilação. O JavaScript também é um script do lado do cliente, acelerando a execução do programa, pois economiza o tempo necessário para se conectar ao servidor;
2ª Simplicidade: JavaScript é fácil de entender e aprender. A estrutura é simples para os usuários, bem como para os desenvolvedores. Também é muito viável de implementar, economizando muito dinheiro para os desenvolvedores desenvolverem conteúdo dinâmico para a web;
3ª Popularidade: Como todos os navegadores modernos suportam JavaScript, ele é visto em quase todos os lugares.

3 Limitações:
1ª Segurança do lado do cliente: Como o código JavaScript é visível para o usuário, outros podem usá-lo para fins maliciosos. Essas práticas podem incluir o uso do código-fonte sem autenticação. Além disso, é muito fácil colocar algum código no site que comprometa a segurança dos dados no site;
2ª Suporte ao navegador: O navegador interpreta o JavaScript de forma diferente em diferentes navegadores. Assim, o código deve ser executado em várias plataformas antes de ser publicado. Os navegadores mais antigos não suportam algumas novas funções e precisamos verificá-las também;
3ª Falta de facilidade de depuração: Embora alguns editores HTML suportem depuração, não é tão eficiente quanto outros editores como editores C/C++. Além disso, como o navegador não apresenta nenhum erro, é difícil para o desenvolvedor detectar o problema.

6 grandes projetos que utilizam a tecnologia: Facebook, Google, Yahoo!, Amazon, Netflix, Paypal.

Nome tecnologia 5: PHP-com-XAMPP
Site oficial da tecnologia:https://https://www.php.net/

Definição da tecnologia: O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.

3 prós de se usar esta tecnologia:
1ª Não é difícil aprender a codificar na plataforma, pois ela possui nomenclaturas já utilizadas no Perl, C e Java, que são linguagens de conhecimento do grande público;
2º Muito versátil, PHP funciona em qualquer plataforma, rodando em diversos sistemas operacionais;
3ª Ele possui código aberto, tratando-se de uma linguagem gratuita onde são feitas melhorias constantemente.

3 Limitações:
1ª Não há uma padronização entre as suas versões, nem todos os comandos funcionam da mesma forma;
2ª A falta de aplicativos de servidor para PHP é algo que desaponta, por isso alguns preferem utilizar o Java;
3ª Os documentos demoram para serem adicionados no site oficial, as novas funções são feitas e depois divulgadas.

6 grandes projetos que utilizam a tecnologia: Facebook, Google, 123Milhas, Wikipedia, Spotify, Uber.
